lazySlider How-to

首先引入lazySlider的样式表

            
                <link rel="stylesheet" href="css/lazySlider-default.css">

然后在html中放置一个要装slider的容器比如#slider

            
                <div id="slider"></div>

引入lazySlider.js 然后参照如下配置对lazySlider进行初始化

            
                <script src="js/lazySlider.js"></script>
                <script>
                    //初始化lazySlider
                    lazySlider('#slider', //承载容器
                            {
                                config: {width: 900, height: 300, autoplay: true,delay:3000},//配置
                                data: [
                                    {img: '1.jpg', alt: 'words', h1: 'Super Lazy', h2: 'Slider'},//数据
                                    {img: '2.jpg', alt: 'words', h1: 'Friendly', h2: 'DEVIL'},
                                    {img: '3.jpg', alt: 'words', h1: 'Creative', h2: 'JIKE'},
                                    {img: '4.jpg', alt: 'words', h1: 'Super Lazy', h2: 'Slider'},
                                    {img: '5.jpg', alt: 'words', h1: 'Creative', h2: 'DUET'},
                                    {img: '6.jpg', alt: 'words', h1: 'Creative', h2: 'JIKE'}
                                ]
                            }
                    );
                </script>

目前可用配置项:

            
                参数:
                    width  //宽度 不加px
                    height //高度 不加px
                    autoplay // true为开启自动播放 false关闭 默认关闭
                    delay //自动播放延时 单位毫秒 默认2000
                数据:
                    img  //图片地址
                    alt  //图片的alt
                    h1   //caption部分的文字
                    h2   //caption部分的文字
